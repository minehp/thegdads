Title atau judul dalam bagian header website berguna untuk memberi informasi ke pembaca tentang isi materi yang akan user baca atau akses. Sedangkan metadata berguna untuk memberi informasi ke robot pencari (search engine) agar website yang kita buat bisa index dan dimasukkan ke database search engine seperti google, bing, duckduckgo dan lain-lain. Angular adalah javascript framework yang dapat digunakan untuk membuat static ataupun dinamis website.

Apakah perlu? dan mengapa harus dinamis?

Static website yang digunakan untuk web profil atau blog sangat membutuhkan title dan metadata. Tujuan dari title dan metadata ini agar search engine dengan mudah melakukan indexing sehingga website bisa masuk dalam daftar pencarian user. Meskipun secara teknis implementasi-nya title dan metadata ini tidak bisa secara cepat membuat website langsung diindex dan dibaca oleh search engine, tetapi saat search engine bisa mendeteksi website yang kita buat, search engine bisa dengan cepat, mudah dan tanpa kendala memasukkan website ke database mereka.

Alasan utama dinamik title dan metadata untuk sebuah website adalah konten yang akan di sediakan sebuah website tersebut bukan konten statis, seperti blog atau web profil yang tiap beberapa periode sekali terjadi perubahan konten. Dinamik title dan metadata
